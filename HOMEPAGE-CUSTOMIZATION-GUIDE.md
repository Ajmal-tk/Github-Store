# Gadget Hub Homepage Customization Guide

Your homepage is now **fully customizable** through the WordPress admin panel! No coding required.

## How to Customize Your Homepage

### Step 1: Access the Customizer

1. Log in to your WordPress admin panel
2. Go to **Appearance → Customize**
3. Look for the **"Gadget Hub Homepage"** panel

### Step 2: Customize Each Section

The customizer is organized into easy-to-use sections:

---

## 📢 Top Strip Banner

**Path:** `Appearance → Customize → Gadget Hub Homepage → Top Strip Banner`

- **Strip Text**: The promotional banner at the very top of your homepage
- Default: "⚡ Festive Mega Sale · Extra 10% on prepaid · Free returns"
- You can use emojis and special characters!

---

## 🎯 Hero Section

**Path:** `Appearance → Customize → Gadget Hub Homepage → Hero Section`

Customize the large hero banner at the top:

- **Hero Pill Text**: Small badge text (e.g., "New Season · Lightning Deals")
- **Hero Title**: Main heading (e.g., "Top gadgets picked for you")
- **Hero Description**: Subtitle/description text
- **Hero Button Text**: Call-to-action button text (e.g., "Shop all deals →")
- **Hero Button URL**: Where the button links to (default: /shop/)
- **Hero Image**: Upload your own hero image or use a URL

### Hero Features (3 feature boxes below hero)

- **Feature 1-3 Icon**: Emoji or icon (e.g., 🚚, 🛡️, 💳)
- **Feature 1-3 Text**: Main feature text (e.g., "Same-day dispatch")
- **Feature 1-3 Subtext**: Additional info (e.g., "Metro cities")

---

## 🛍️ Product Sections

**Path:** `Appearance → Customize → Gadget Hub Homepage → Product Sections`

Control how products are displayed:

- **First Product Section Title**: Heading for first product grid (default: "Top deals for you")
- **Number of Products (Section 1)**: How many products to show (default: 8)
- **Second Product Section Title**: Heading for second product grid (default: "More Products")
- **Number of Products (Section 2)**: How many products to show (default: 8)

**Note:** Products are automatically pulled from your WooCommerce products based on:
- Latest products first
- Only in-stock items
- Featured products get a "Hot" badge
- Products on sale get a "Sale" badge

---

## 🎨 Middle Banner

**Path:** `Appearance → Customize → Gadget Hub Homepage → Middle Banner`

The promotional banner between product sections:

- **Show Banner Section**: Toggle to show/hide this entire section
- **Banner Section Title**: Section heading (e.g., "Big-screen deals")
- **Section Link Text**: Text for "View all" link (e.g., "View TVs")
- **Banner Title**: Main banner heading (e.g., "65" 4K QLED TVs from $699")
- **Banner Description**: Banner description text
- **Banner Button Text**: Button text (e.g., "Explore offers")
- **Banner Button URL**: Where the button links to
- **Banner Image**: Upload a custom banner image

---

## 📝 Managing Products

Products shown on the homepage come from your **WooCommerce products**:

### To Update Products:

1. Go to **Products → All Products** in WordPress admin
2. Add/Edit products:
   - **Product Name**: Shows as title on homepage
   - **Featured Image**: Shows as product image
   - **Short Description**: Shows as product description (first 8 words)
   - **Price**: Shows with currency
   - **Sale Price**: Automatically adds "Sale" badge
   - **Featured Product**: Adds "Hot" badge

### Product Badges:

- **"Sale"**: Automatically appears when product has a sale price
- **"Hot"**: Automatically appears when product is marked as featured
- **"Featured"**: Shows in second product section

## 💡 Tips & Best Practices

### Images:
- **Hero Image**: Recommended size 900x600px
- **Banner Image**: Recommended size 1000x600px
- **Product Images**: Recommended size 800x800px (square)

### Text:
- Keep hero title short and punchy (under 50 characters)
- Hero description should be 1-2 sentences
- Use emojis sparingly for visual interest (🚚 ⚡ 🎉)

### Products:
- Add at least 8 products to fill the first section
- Add 16+ products to show both sections
- Mark bestsellers as "Featured" to highlight them
- Use sale prices to create urgency

## 🆘 Troubleshooting

**Products not showing?**
- Make sure products are Published (not Draft)
- Check that products are marked "In Stock"
- Verify you have WooCommerce installed and activated

**Images not displaying?**
- Use direct image URLs or upload via Media Library
- Check image file size (keep under 2MB for performance)
- Ensure images are accessible (not blocked by security)

**Changes not saving?**
- Click "Publish" button in Customizer (top left)
- Clear browser cache (Ctrl+Shift+R)
- Clear WordPress cache if using a cache plugin
